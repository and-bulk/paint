# Init project

## In client dir

> Set env vars and install packages

```
cp .env.example .env
npm ci
```

> Compiles and hot-reloads for development

```
npm start
```

> Compiles and minifies for production

```
npm run build
```

> Run tests

```
npm run test
```

## In server dir

> Set env vars and install packages

```
cp .env.example .env
npm ci
```

> Compiles and hot-reloads for development

```
npm run dev
```

> Compiles and minifies for production

```
npm run build
```

> Run nodejs server

```
npm start
```
